The encryption work factor can be adjusted by this option.

sensitive kinds of presets generally enable the highest work factor, so it's the most secure option from the list, but its execution takes significantly longer than other kinds of presets. moderate is in the middle and interactive is the least secure option, but it works faster.

sodium.crypto_pwhash is considered as a more secure key derivation function than node.pbkdf2 as it goes with the Argon2 algorithm.
